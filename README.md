# FOBU-MC-Anticheat
Fate Of Beyond Universe Minecraft Anticheat made for private servers for fun

You can download an builded jar in release tag.

Don't forget to setup the config fobu_mc_anticheat or the anticheat won't work.

This Anticheat kick the player with an random message(kick_message.txt) if the player have an mod/pack which includes one of the banned words(banned_mods.txt/banned_packs.txt) in it's json id inside the [mod].jar or if the player have a different anticheat version as server.

The fatal flaw is well known with the way the anticheat works
